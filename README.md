# 个人简历 / Personal Resume

本项目是我的个人简历源代码仓库。简历采用纯 HTML/CSS 编写，旨在直接通过浏览器打印生成高质量的 PDF 文件。

## 📖 如何使用

1. 克隆本项目到本地：
   ```bash
   git clone <你的仓库地址>
   ```
2. 使用现代浏览器（如 Chrome、Edge 或 Safari）打开仓库中的 HTML 文件（例如 `resume_en_academic.html`）。
3. 按下快捷键 `Ctrl + P` (Windows/Linux) 或 `Cmd + P` (Mac) 打开浏览器的打印预览面板。
4. **推荐的打印设置**：
   - **目标打印机 (Destination)**：另存为 PDF (Save as PDF)
   - **边距 (Margins)**：无 (None) —— 这能保证设计好的排版填满页面
   - **背景图形 (Background graphics)**：✅ 勾选 —— 开启此项以确保背景颜色和设计元素正常显示
   - **页眉和页脚 (Headers and footers)**：❌ 取消勾选 —— 防止浏览器自动添加网址、日期等默认信息
5. 点击“保存”，即可生成最终用于投递的 PDF 格式简历。
